注：	1、使用SDK开发包前，建议先用客户端软件调试，显示正常再二次开发。

	2、电脑需安装visual studio 2008运行环境，如没有安装，文件夹【visual studio 2008运行环境】有安装包
	
Note：
	1、Before using the SDK development kit, it is recommended to debug with the client software first, 
	   display it normally, and then develop it again
	   
	2、The computer needs to install the visual studio 2008 operating environment, if not installed, 
	     there is an installation package in the folder [visual studio 2008 operating environment]


文件夹说明（Folder description）：

Demo(C#)      		1、32位的C#演示程序，如需64位请到文件夹【DLL(Dynamic library)】复制替换；
                     		2、构建用的是Visual Studio 2017版本，低版本请自行移植；

Demo(java)    		1、64位的java演示程序，如需32位请到文件夹【DLL(Dynamic library)】复制替换；
	      
Demo(VC)      		1、32位的VC演示程序，如需64位请到文件夹【DLL(Dynamic library)】复制替换；

DLL(Dynamic library)    	里面包括32位与64位的Unicode和多字节字符集版本

visual studio 2008运行环境	DLL相关运行库，如没有安装，请安装才可正常调用DLL
